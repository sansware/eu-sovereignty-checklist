# Contributing

This is a checklist; contributions welcome via PR. Tag [@sansware](https://github.com/sansware)
for review.

A few notes that save round-trips:

- **Scope.** The repository is intentionally small. New checks should map
  cleanly to one of the five principles in [`README.md`](README.md), or
  expand the wider [`CHECKLIST.md`](CHECKLIST.md). Out-of-scope ideas
  are better as a fork.
- **Tone.** Engineering-hygiene voice; no marketing wedge.
- **Licence.** Contributions are accepted under
  [CC BY 4.0](LICENSE), the same licence as the rest of the project.
- **Conduct.** All contributors agree to the
  [Code of Conduct](CODE_OF_CONDUCT.md).
- **Security issues.** Do not open a public PR or issue for a
  vulnerability — follow [`SECURITY.md`](SECURITY.md) instead.
